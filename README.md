# ナレッジグラフ推論チャレンジ プログラム、データ等の提出物
 - チーム名: 上小田中411
## 提出物のディレクトリ構造
  - src – ナレッジグラフを操作するコマンドのソース
  - build     - ナレッジグラフを操作するコマンドの実行系
  - data - 中間生成物, グラフ埋め込みデータ
  - dict – 単語埋め込みデータ、分かち書き等の辞書
  - gradlew - ソースをコンパイルするコマンド
  - Makefile - 容疑者を検出するスクリプト
  - Form.pdf - 応募フォーム
